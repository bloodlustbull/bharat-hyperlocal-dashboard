AI-powered city-language growth engine for Indian quick-commerce brands.

The dashboard helps teams select city-language-category wedges, convert them into local content briefs, plan WhatsApp funnel experiments, and track weekly optimization once real pilot campaign data is uploaded.

city-language scorer → ICP selector → trend scraper → content brief generator → WhatsApp funnel → experiment tracker → weekly optimization dashboard

Engine flow

How the GTM engine operates

01City-language scorer

Ranks local opportunity wedges using the seed model.

02ICP selector

Frames the priority audience and category use case.

03Trend scraper

Placeholder operating stage for future local trend inputs.

04Content brief generator

Turns the selected wedge into a manager-ready brief.

05WhatsApp funnel

Structures trigger, message, offer, and conversion path.

06Experiment tracker

Reads uploaded pilot CSV data in-browser.

07Weekly optimization

Uses real pilot results once available.
